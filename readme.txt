This plugin creates a simple form on the front end that allows you to keep a log of visitors, limiting the number of times per day any one can visit.

== Installation ==

1. Upload the `checkinator` directory to `/wp-content/plugins/`
1. Activate the plugin through the 'Plugins' menu in WordPress
